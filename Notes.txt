Setup Development Environment 

On the Windows Server:
1. Setup kubectl and kubeconfig as described in Module 1's exercise
2. Download and install Visual Studio Code
3. Install the Kubenetes extension by Microsoft
4. Optionally, install the YAML extension by Red Hat
5. Install Git from https://git-scm.com/downloads
6. Ctrl-Shift P and then type Git: Clone
7. Clone the repository from:  https://github.com/netapp-devops/course
