Setup Development Environment 

On the Windows Server:
1. Setup kubectl and kubeconfig as described in Module 2's exercise 
2. Install Git for Windows from https://git-scm.com/downloads
3. Download and install Visual Studio Code
4. Install the Kubenetes extension by Microsoft
5. Ctrl-Shift P and then type Git: Clone
6. Clone the repository from:  https://github.com/netapp-devops/course
